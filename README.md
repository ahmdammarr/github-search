## Setup

- Create [Github Auth App](https://docs.github.com/en/apps/oauth-apps/building-oauth-apps/creating-an-oauth-app)
- Add the following variables to the `.env` file

```sh
client_id=
redirect_uri=
login=
scope=
state=
allow_signup=
access_token= // To be removed after auth feature
```
